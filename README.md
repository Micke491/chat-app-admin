## Admin Panel

The project includes a comprehensive Admin Panel for platform management.

### Accessing the Admin Panel
1. Navigate to `/admin` in your browser.
2. The panel is protected by authentication. Only users with the `admin` role can access it.
3. If you are not logged in or do not have admin permissions, you will be redirected to the home page.

### Features
- **Dashboard**: Real-time overview of platform statistics, including total users, messages, and active stories.
- **User Management**: Monitor user growth, manage user status, and handle bans.
- **Message Control**: View platform-wide message activity and manage content.
- **Story Management**: Overview of active and total stories shared by users.
- **Reports**: Dedicated section for handling user-submitted reports and maintaining platform safety.
